# Credit Risk Assessment System

This project implements an end-to-end, explainable machine learning system to assess loan default risk.
It combines robust ML modeling with transparent decision support and a production-ready Streamlit interface.

## Objectives
- Predict probability of loan default
- Convert predictions into business-friendly risk scores
- Provide explainability for trust and transparency
- Demonstrate responsible AI practices

## Key Features
- Unified sklearn Pipeline with XGBoost
- Explainable predictions (SHAP-lite)
- Streamlit decision-support UI
- Session-based prediction history
- Model governance and ethical disclaimers
